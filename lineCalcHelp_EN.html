Info

LineCalc 0.2 (beta) - compiled with Java 14

LineCalc is a simple, command-line calculator. It features a basic variable management system and recognizes several mathematical functions. It is intended for quick, back-of-the-envelope calculations and for aid in classroom problems.

It is designed as a stand-alone Java application.

How to use

Just type the math expression you want to evaluate into the command line (the text field at the top of the window) and hit Enter. The result will appear in the text field just below the command line.

The program supports a wide range of functions besides the basic operations. The complete list is given below.

Multiplications must be entered explicitly with * : inserting a blank space between two numbers/variables/constants will result in an error message.

Complex numbers are supported; they have to be entered as the sum of their real and imaginary part, using the built-in constant i (e.g. 2+3*i).

Function arguments must be surrounded by round brackets () and separated by commas , .

Variable management

The app can manage user-defined variables. The current list of variables is shown in the table below the three buttons. Note: built-in constants (like π or e) are not shown in the app window, but are listed in this help page. Both variables and constants may be used in an expression.

  • Creates a variable and initializes it with the current value of the expression (if no expression has been evaluated yet, the button does nothing). A popup window will appear allowing to choose the variable name. The name must follow a few rules:
    • It must be unique. It cannot be already assigned to another variable or a built-in constant.
    • It must start with a letter (upper- or lowercase).
    • It must contain only letters (upper- or lowercase), digits (0-9) or underscores ( _ ).
  • Deletes the selected variable from the list (if no variable is selected, the button does nothing). A deleted variable can no longer be used in an expression.
  • Assigns the current value of the expression to the selected variable, overriding the previous value (if no variable is selected, the button does nothing).

Factorization

The Factorize button ("Fattorizza") opens a popup window; entering a positive whole number and then pressing the Calculate button (or typing Enter) will show the prime factor decomposition of the number, indicating each factor's multiplicity. Close the window to go back to the main window.
