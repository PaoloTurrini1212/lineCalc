About

LineCalc 0.2 (beta) - compiled with Java 14

LineCalc is a simple, command-line calculator. It features a basic variable management system and recognizes several mathematical functions. It is intended for quick, back-of-the-envelope calculations and for aid in classroom problems.

It is designed as a standalone Java application.

How to use

Just type the mathematical expression you want to evaluate in the command line (the text field at the top) and press Enter. The result will be displayed in the field just below the command line.

There is a wide range of supported functions beyond the basic operations. The full list can be found below.

Multiplications must be explicitly stated with * : inserting a space between numbers/variables/constants will get an error.

Complex numbers are supported; they have to be input as a sum of the real and imaginary parts and using the built-in constant I (e.g. 2+3*I)

Function arguments must be enclosed in parentheses () and separated by commas , . Function names start with a lowercase letter.

Variable management

The app can manage variables. The current variable list is displayed in the table below the three buttons. Note: There are a few built-in constants (like π or e), not displayed in the variable list. Variables and constants can be used inside an expression.

  • The "+" button creates a variable with the current value of the expression (if no expression has been evaluated, the button does nothing). A popup window will ask for the variable name. The name must follow some rules:
    • Be unique. The name cannot be the same as another variable, nor one of the built-in constants.
    • Start with an upper-case letter.
    • Contain only letters (upper- or lower-case), digits (0-9) or underscores ( _ ).
  • The "-" button deletes the selected variable from the list (if no variable is selected, the button does nothing). A deleted variable cannot be used anymore in an expression.
  • The "↓" button assigns the current value of the expression to the selected variable, substituting its previous value (if no variable is selected, the button does nothing).
